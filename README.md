## Heart Disease EDA & Machine Learning (R)

This project explores a heart disease dataset using exploratory data analysis and multiple machine learning models in R.

### Key steps
- Data visualization and summary statistics
- Logistic regression, XGBoost, and random forest models
- Model comparison and interpretation
- PCA, clustering, and correspondence-style analysis

### Tools
- R, ggplot2, dplyr
- caret / randomForest / xgboost
- FactoMineR

### Files
- `analysis.Rmd` – full analysis and code
- `analysis.html` – rendered output
